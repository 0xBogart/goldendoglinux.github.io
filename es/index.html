¿Una Estación de Trabajo Linux Hecha Simple?

GoldenDog Linux es una selección automatizada de paquetes y configuraciones para un Sistema Operativo Debian Estable. Esta es la selección de paquetes que diseñé para uso personal y ahora decidí automatizar y empacar como un archivo ISO instalable.

¿Qué tal Actualizar mi sistema?

Dado que GDL, en su núcleo, utiliza software Debian, y Debian recomienda utilizar sus espejos para garantizar la fiabilidad y estabilidad, GoldenDog utiliza los repositorios oficiales de Debian Estable, manteniendo un rendimiento robusto y fiable, y además, todas las actualizaciones y bibliotecas de software que están disponibles actualmente para Debian Bookworm están disponibles para GoldenDog.

Un lindo Escritorio para uso diario

GoldenDog no depende de flatpak o snap para el software. Viene con algunos software preinstalados, como Visual Studio Code, junto con su repositorio oficial (Microsoft) para que siempre pueda mantenerlo actualizado a su última versión. ¡Muchas aplicaciones se ejecutan nativamente como Steam, kdenlive, obs, Spotify y muchas más! Ideal para trabajar, jugar, desarrollar o revisar el correo y chatear.

GDL nació como una herramienta de automatización para mí mismo, para implementar mi propio sistema operativo de manera consistente en mis diferentes máquinas. Algunas personas escucharon sobre ello y sugirieron que convirtiera mi herramienta en un .iso público instalable para todos los que quieran tener una configuración similar, así que aquí está. Ha sido elaborado con mucho amor. ¡Espero que les guste!
