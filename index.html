A Linux Workstation Made Simple?

GoldenDog Linux is an automated selection of packages and presets for a Debian Stable Operating System. This is the selection of packages that I designed for personal use and now I decided to automatize and pack as an installable iso file.

What about Updating my system?

Since GDL, at its core, uses Debian software, and Debian recommends to use their mirrors to ensure reliability and stability, GoldenDog uses Debian Stable official repositores, maintaining a robust and reliable performance, and also, all updates and software libraries that are currently available for Debian Bookworm are available for GoldenDog

A cute Desktop for everyday use

GoldenDog does not rely on flatpak or snap for software. It comes with some software preinstalled, like Visual Studio Code, along with its official repository (Microsoft) so you can always keep it updated to its latest version. Lots of applications run natively like Steam, kdenlive, obs, Spotify, and many more! Ideal for working, gaming, developing or checking the mail and chatting!

GDL was born as an automation tool for myself to deploy my own operating system consistently across my different machines. Some people heard about it and suggested I turned my tool into an installable public .iso for everyone who wants to have a similar setup, so here it is. It's been handcrafted with lots of love. Hope you like it!
